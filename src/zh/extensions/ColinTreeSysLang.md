## 方法

* 获取系统当前语言的编码
  {% Ai2Method %}{"name":"SysLang", "output":true}{% endAi2Method %}
* 系统语言是否为英文
  {% Ai2Method %}{"name":"SysLangEn", "output":true}{% endAi2Method %}
* 系统语言是否为中文
  {% Ai2Method %}{"name":"SysLangZh", "output":true}{% endAi2Method %}
